Everything you need to ship

Skip months of boilerplate. PromptForge gives you a solid foundation — designed and architected so you can build fast without cutting corners.

⬛
Next.js 16 App Router
Nested layouts, React Server Components, streaming, and parallel routes — all pre-configured and ready to extend.
🎨
Custom Design System
Three bespoke Tailwind color families — ink, accent, ember — with custom shadows and animation keyframes baked in.
🟦
TypeScript Strict Mode
Full type-safety from day one. @/* path aliases configured, incremental builds enabled.
🌙
Dark Mode First
Class-based dark mode with no flash. Designed dark from the ground up — light mode is a first-class citizen too.
⚡
Optimised Build Pipeline
ESLint 9 flat config, PostCSS + Autoprefixer, and npm ci locked installs for reproducible deployments.

Lucide Icons + clsx
Tree-shakable icon library and a battle-tested class name utility — two essential companions for building fast UI.

// quickstart

Up and running in 60 seconds

Clone, install, configure — that's it. The dev server has hot reload and TypeScript errors in your terminal.

1
Clone & install 
# clone the repo
git clone https://github.com/aryanraj6308/promptforge-ai-saas
cd promptforge-ai-saas

# locked, reproducible install
npm ci
2
Set environment variables 
cp .env.example .env.local

# required keys in .env.local:
NEXT_PUBLIC_APP_URL=http://localhost:3000
ANTHROPIC_API_KEY=sk-ant-…
DATABASE_URL=postgres://…
NEXTAUTH_SECRET=your-secret
3
Start the dev server 
npm run dev

# open in your browser:
http://localhost:3000

# production build:
npm run build && npm start
4
Deploy to Vercel 
# install Vercel CLI globally
npm i -g vercel

# deploy with one command
vercel

# or push to main on GitHub —
# Vercel auto-deploys on push
